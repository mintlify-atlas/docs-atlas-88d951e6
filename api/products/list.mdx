## Overview

Retrieve a list of all products in your environment. This endpoint returns all product configurations including active, archived, and default products with their complete item structures.

<CodeGroup>
```bash cURL
curl https://api.useautumn.com/products \
  -H "Authorization: Bearer YOUR_SECRET_KEY"
```

```javascript JavaScript SDK
import { Autumn } from '@useautumn/sdk';

const autumn = new Autumn({
  secretKey: 'YOUR_SECRET_KEY'
});

const { list } = await autumn.products.list();
```

```typescript TypeScript SDK
import { Autumn } from '@useautumn/sdk';

const autumn = new Autumn({
  secretKey: 'YOUR_SECRET_KEY'
});

const { list } = await autumn.products.list();
```
</CodeGroup>

<ResponseField name="list" type="array">
  Array of product objects. Each product contains complete configuration details.
  
  <Expandable title="Product Object Properties">

    </ResponseField>
    
    <ResponseField name="name" type="string">
      The display name of the product.
    </ResponseField>
    
    <ResponseField name="description" type="string">
      The product description.
    </ResponseField>
    
    <ResponseField name="group" type="string">
      The product group identifier. Products in the same group follow upgrade/downgrade logic.
    </ResponseField>
    
    <ResponseField name="env" type="string">

    </ResponseField>
    
    <ResponseField name="is_add_on" type="boolean">

## Use Cases

### Display Pricing Page

Retrieve all products to build your pricing page:

```javascript
const { list: products } = await autumn.products.list();

// Filter out archived products
const activeProducts = products.filter(p => !p.archived && !p.is_add_on);

// Sort by price
activeProducts.sort((a, b) => {
  const priceA = a.items.find(i => i.type === 'price')?.price || 0;
  const priceB = b.items.find(i => i.type === 'price')?.price || 0;
  return priceA - priceB;
});

// Display products
activeProducts.forEach(product => {
  console.log(`${product.name}: $${product.items[0].price}/${product.items[0].interval}`);
});
```

### Filter by Product Type

Get specific types of products:

```javascript
const { list: products } = await autumn.products.list();

// Get add-ons only
const addons = products.filter(p => p.is_add_on);

// Get subscription plans only
const subscriptions = products.filter(p => 
  !p.is_add_on && 
  !p.properties?.is_free && 
  !p.properties?.is_one_off
);

// Get products with free trials
const trialProducts = products.filter(p => p.free_trial !== null);
```

### Group by Interval

Organize products by billing frequency:

```javascript
const { list: products } = await autumn.products.list();

const byInterval = products.reduce((acc, product) => {
  const interval = product.properties?.interval_group || 'one-time';
  if (!acc[interval]) acc[interval] = [];
  acc[interval].push(product);
  return acc;
}, {});

console.log('Monthly plans:', byInterval.month?.length || 0);
console.log('Annual plans:', byInterval.year?.length || 0);
```

### Find Default Product

Get the default product for new customers:

```javascript
const { list: products } = await autumn.products.list();
const defaultProduct = products.find(p => p.is_default);

if (defaultProduct) {
  console.log('Default product:', defaultProduct.name);
} else {
  console.log('No default product set');
}
```

### Check Product Versions

Find products with multiple versions:

```javascript
const { list: products } = await autumn.products.list();

const productVersions = products.reduce((acc, product) => {
  if (!acc[product.id]) {
    acc[product.id] = [];
  }
  acc[product.id].push(product.version);
  return acc;
}, {});

// Find products with multiple versions
Object.entries(productVersions).forEach(([id, versions]) => {
  if (versions.length > 1) {
    console.log(`${id} has ${versions.length} versions:`, versions);
  }
});
```

### Build Product Comparison

Create a feature comparison matrix:

```javascript
const { list: products } = await autumn.products.list();

const comparison = products.map(product => ({
  name: product.name,
  price: product.items.find(i => i.type === 'price')?.price || 0,
  features: product.items
    .filter(i => i.feature_id)
    .map(i => ({
      id: i.feature_id,
      included: i.included_usage,
      unlimited: i.included_usage === 'infinite'
    }))
}));

console.table(comparison);
```
